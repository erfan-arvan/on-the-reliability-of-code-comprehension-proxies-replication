# Open Coding

Materials from the open coding process used to build the taxonomy of code comprehension task types (Figure 1 and Figure 2 in the paper).

| File | Description |
|---|---|
| `FinalTreeWithNumbers.txt` | Final taxonomy tree (Figure 1) — 111 leaf nodes across 4 top-level categories. Printed to terminal by `python show_figure1.py` |
| `open_coding_logs.xlsx` | Round-by-round log of coding decisions, merges, and label refinements made during the open coding sessions |
| `kappa_open_coding.py` | Computes inter-rater agreement (Cohen's kappa) between the two coders. Run with: `python open_coding/kappa_open_coding.py` |
