# Study Materials

All instruments used in the study, in both human-readable and machine-readable form.

| File | Description |
|---|---|
| `snippets_and_tasks.md` | The 8 Java code snippets shown to participants, together with the comprehension tasks (output prediction, syntax/binary, function description) |
| `familiarity_survey.md` | CS knowledge background survey shown before the study, with correct answers marked |
| `experience_survey.md` | Java experience questionnaire |
| `selected_methods.csv` | All candidate methods considered before final selection |
| `repositories.csv` | Open-source repositories from which methods were drawn |
| `question_design_process_and_prompts_used.txt` | Description of the question design process and LLM prompts used to generate initial question drafts |
