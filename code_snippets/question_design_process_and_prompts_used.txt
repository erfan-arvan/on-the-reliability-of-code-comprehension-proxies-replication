Design questions

To design questions based on the selected subcategories of tasks, we developed a clear and semi-reproducible procedure–unlike many previous studies, which often provided little to no detail about how their comprehension questions were constructed and thus offered no replicable methodology.
 
Our process began by compiling the definition of each selected subcategory, along with sample questions drawn from prior studies. This documentation was included in an explanatory prompt for ChatGPT, which we then used to generate candidate questions for each of the 8 code snippets. The prompt was designed to mimic the style and structure of prior work in the literature.

Example prompts:
I'm working on a study involving 8 code snippets, each of which is a small real-world Java method, and I need to create comprehension questions for each one. These questions fall into different question categories, such as Program Function, Semantics, Syntax, and Other, with more specific subcategories (e.g., Simulated Execution -> Internal State with Specific Input). I'll start by sending you one snippet along with a specific question category and a few example questions. For each example that's applicable to the snippet, please generate three candidate questions in that same style. If an example is a yes/no question, then provide three candidates with the answer "Yes" and three with the answer "No." Note: These questions are being designed for a code comprehension study involving undergraduate computer science students. Please ensure the questions are suitable for that audience. Let me know if anything is unclear. I’ll send the first Java method snippet and question category next.
Here’s the first question category: "Semantics -> Program Analysis -> Semantics - Count" This category focuses on questions that require counting or evaluating quantities based on the program’s logic and behavior. 
Example questions from this category (drawn from various programming languages): 

"What is the maximum number of references (fetch or store) to the array CASH that can occur during a call to the function SAFE??
