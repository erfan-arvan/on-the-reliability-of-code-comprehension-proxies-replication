# Gradings

Materials from the manual grading of participant responses to the function-description task.

Each response was graded 0–3 by two independent raters, then normalized to 0–1 for analysis.

| File | Description |
|---|---|
| `function_question_grading_log.xlsx` | Full grading log: both raters' scores for every participant × snippet response |
| `kappa_for_gradings.py` | Computes inter-rater agreement (Cohen's kappa) for the function-description grading. Run with: `python gradings/kappa_for_gradings.py` |
